<b>Requires:</b> [.NET Framework 4.7.2](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net472 ".NET Framework 4.7.2") and v143 Build tools.

<b>How to use:</b><br>
- Disable Antivirus
- Run CS2
